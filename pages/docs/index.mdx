## Quick Start Guide

1. **Create an API Key**  
   Generate your API key by visiting [makehub.ai/api-keys](https://makehub.ai/api-keys)

2. **Use Our Endpoint**  
   Connect to our OpenAI-compatible endpoint:
   ```
   https://api.makehub.ai/v1
   ```

3. **Explore Available Models**  
   Browse our models at [makehub.ai/models](https://makehub.ai/models)
